# RitzCahors_eval

Git clone sur ce repository.
Pour pouvoir utiliser sa propre base de données, changer les informations relatives à la base de données dans le fichier "database.php". Elles se trouvent dans la classe ConnectDB dans la fonction construct :
$connec_dbname = '', $connec_pseudo = '', $connec_mdp = ''

Ensuite dans le dossier sur le terminal, taper la commande : php -S localhost:8000
